# Aviation Accident Analysis

This is a beginner-level data analysis project exploring patterns in aviation accidents using a dataset from Kaggle.

##  Project Objectives

- Clean and prepare raw aviation data
- Explore accident trends using Python (Pandas + Matplotlib)
- Visualize key patterns such as fatalities, flight phases, weather conditions, and year-wise trends
- Practice exploratory data analysis (EDA) and data cleaning

##  Files & Structure

- `notebooks/` — Jupyter notebook(s) for cleaning and EDA
- `data/raw/` — Raw dataset (original Kaggle data)
- `data/clean/` — Cleaned dataset after preparation
- `images/` — Visualizations and plots
- `README.md` — This project overview
- `.gitignore` — To exclude unnecessary files (e.g., temp, cache)

## 🛠️Tools Used

- Python 3.x
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook
- Git & GitHub

## 📊Data Source

**Dataset**: [Aviation Accident Data](https://www.kaggle.com/datasets)  
*Note: This project is for learning purposes and does not reflect official aviation records.*
